## ipam dhcp接口文档



## 全局设置：

- 所有的变量类型都是字符串或整数
- 接口暂定，联调时候继续优化。



### 1 子网管理

#### 1.1 添加子网

| 功能     | 描述                                                         |
| -------- | ------------------------------------------------------------ |
| 接口功能 | 获取top域名和top ip                                          |
| 接口地址 | /apis/linkingthing.com/example/v1/subnetv4s                  |
| 请求方式 | POST                                                         |
| 请求参数 | 无                                                           |
|          | name, 类型 string                                            |
|          | gateway, 类型 string //暂时不生效，后期补充                  |
| 请求示例 | curl http://10.0.0.101:8081/apis/linkingthing.com/example/v1/subnetv4s -X POST -d '{"subnet":"10.0.1.0/24", "name":"name2"}' |
